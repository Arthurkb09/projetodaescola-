Exemplos de editores de texto WYSIWYG

São editores que permitem que vejas como o código aparece numa página web em tempo real. Para os iniciantes, os editores WYSIWYG são altamente recomendados, pois não é necessário ter experiência em codificação.

Alguns Editores WYSIWYG populares:

  1. Atom
  2. Sublime Text
  3. Notepad++
  4. Brackets
